programming language features have:
	SYNTAX: the grammer, how you say it
	SEMANTICS: the meaning, what it does.
the syntax of a print statement(technically, a call function named 'print') is 
	print (LIST-OF-ARGUEMENTS )
the semantics are: print out the value of each argument in the list.
the syntax of an assignment statement is
	VARIABLE_NAME = EXPRESSION
